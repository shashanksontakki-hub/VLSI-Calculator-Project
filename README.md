# VLSI-Calculator-Project
Digital Logic Implementation of a 4-bit Calculator from RTL to GDSII.
<br>
Author - Shashank Sontakki
<br>
<br>
Project Structure
<br>
RTL/: Contains the source Verilog files.
<br>
rca_4bit.v: 4-bit Ripple Carry Adder.
<br>
alu_4bit.v: Arithmetic Logic Unit (Addition, Subtraction, AND).
<br>
Testbench/: Contains simulation files for verification.
<br>
rca_tb.v: Testbench for the Adder.
<br>
alu_tb.v: Testbench for the ALU.
